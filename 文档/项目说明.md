# 项目说明

## 研究主题

本项目研究结构化电子病历数据在脑卒中超早期预后预测中的建模价值。任务目标是在入院或入 ICU 后早期窗口内，根据人口学、合并症、实验室检查、生命体征及血流动力学衍生特征，预测患者预后是否不良。

## 原始数据来源

原始数据来自 PhysioNet：

- MIMIC-III Waveform Database Matched Subset v1.0：`https://physionet.org/content/mimic3wdb-matched/1.0/`
- MC-MED v1.0.0：`https://physionet.org/content/mc-med/1.0.0/`

当前 `数据/` 目录下的 CSV 是建模用派生表。MIMIC matched subset 主要提供与临床数据库匹配的波形和数值记录；预后 label 需要结合对应临床表、死亡信息、出院去向或病例轨迹文本。MC-MED 的 label 抽取则应优先使用 visit disposition、转归字段和文本证据。

## 任务定义

预测任务为二分类：

- `LABEL=0`：预后良好。
- `LABEL=1`：预后不良。

项目资料中的统一表述为：将病例级预后标准化为 favorable 与 unfavorable 两类。MIMIC 标签来自匹配临床信息、死亡信息、出院去向或病例级轨迹标注；MC-MED 标签可由 ED disposition、转归文本和人工复核规则近似映射。

## 方法概要

1. 构建 6 小时早期观察窗口。
2. 提取人口学、合并症、实验室检查和生命体征特征。
3. 对缺失值、异常值和衍生变量进行处理。
4. 采用 Boruta 与 LASSO 进行特征筛选。
5. 建立 Logistic 回归、随机森林、XGBoost、LightGBM 模型。
6. 使用 MIMIC 进行内部交叉验证。
7. 使用 MC-MED 进行冻结外部验证。
8. 使用 ROC、PR、F1、Brier、校准曲线和 SHAP 解释模型。
