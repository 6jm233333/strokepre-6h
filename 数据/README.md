# 数据目录说明

包含脑卒中超早期预后预测建模所需的结构化电子病历派生特征表。CSV 文件是从 PhysioNet 原始数据源清洗、关联、特征工程和标签抽取后得到的建模表。

## 原始数据来源

| 数据源 | 版本 | 地址 | 用途 |
| --- | --- | --- | --- |
| MIMIC-III Waveform Database Matched Subset | 1.0 | https://physionet.org/content/mimic3wdb-matched/1.0/ | 内部队列的原始监测数据来源之一。预后 label 需结合匹配的 MIMIC-III 临床信息、死亡信息、出院去向或病例轨迹证据抽取。 |
| MC-MED | 1.0.0 | https://physionet.org/content/mc-med/1.0.0/ | 外部验证队列的原始急诊多模态数据来源。预后 label 需结合 visit disposition、转归、文本证据或人工整理规则抽取。 |
| 本目录 CSV | 派生表 | 本地文件 | 训练、验证和复现实验所用的 6h 特征表。 |

## 标签字段

| 字段 | 说明 |
| --- | --- |
| `LABEL` | 二分类建模标签。`0` 表示预后良好，`1` 表示预后不良。 |
| `BINARY_PROGNOSIS` | 标签文本说明。MIMIC 中包括 `Improved`、`Worsened_or_Died`；MC-MED 中包括 `Improved`、`Worsened_or_Severe`。 |
| `trend_label` | MIMIC 中保留的趋势标签；MC-MED 当前为空。 |

## Label 抽取说明

本项目的 `LABEL` 是二分类预后标签，不应由早期窗口内的生命体征、实验室指标或模型输出反推。推荐抽取顺序为：

1. 死亡字段、死亡时间、明确死亡文本。
2. 出院去向或 ED disposition。
3. 病程记录中的改善、恶化、严重残疾、临终关怀、转院、康复/长期照护等证据。
4. 人工复核冲突或证据不足样本。

可直接使用的抽取提示词见 `../代码/脑卒中预后标签提取提示词.md`。

## 特征字段

数据主要包括以下变量类型：

- 人口学特征：年龄、性别。
- 合并症特征：高血压、糖尿病、高脂血症、缺血性心脏病、慢性肾病、房颤、心衰。
- 实验室指标：血糖、肌酐、血细胞比容、血红蛋白、血小板、钾、钠、尿素氮、白细胞、碳酸氢盐、氯等。
- 生命体征：舒张压、收缩压、心率、呼吸频率、外周血氧饱和度、体温等。
- 衍生特征：极差、脉压、平均动脉压、休克指数。
